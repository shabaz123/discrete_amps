# Discrete Preamps

This repository contains some audio preamplifier circuits. The CAD files (schematic and PCB) can be opened with KiCad 10.

Gerber files and PDF copies are also available in the repository, for non-KiCad users.

## Preamp K1
Designed by M. Kellett, intended for a HF receiver for speech (not music). This preamplifier offers about 28 dB of gain, and -3 dB bandwidth from about 150 Hz to 10 kHz (very approximate).

## Preamp K2
Designed by M. Kellett, intended for a HF receiver for speech (not music). This preamplifier offers about 45 dB of gain, and -3 dB bandwidth from about 100 Hz to 7 kHz (very approximately).
